Octave¶

Octave is free software to replace Matlab.

start octave¶

run octave-cli

change ps1¶

PS1('>> ')

load/save data¶

load('yourData.dat')

Figure¶

t = [0:0.01:1]; 

y1 = sin(2*pi*t);
plot(t, y1);

y2 = cos(2*pi*t);
plot(t, y2);
plot(x, y, 'rx', 'MarkerSize', 10); % red cross

plot(t, y1);
hold on;                    % don't close figure
plot(t, y2);
xlabel('time')
ylabel('value')
legend('sin', 'cos')
title('my plot')
print -dpng 'myPlot.png'    % save as png

close the figure

close

multi-figures

figure(1): plot(t, y1);
figure(2): plot(t, y2);

subplot

subplot(1, 2, 1);  % create 1x2 figure, use the 1st one
plot(t, y1);

create a file with extension '.m'

function y = square()

end

addpath("path/you/want")
