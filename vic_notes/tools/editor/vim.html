vim¶

vim是一个独特又古老的编辑器，号称最强大的代码编辑工具

安装youcompleteme¶

sudo apt-get install vim-addon-manager
sudo apt-get install vim-youcompleteme
vam install youcompleteme
vim ~/.vimrc

ycm使用全局配置

let g:ycm_global_ycm_extra_conf = '/usr/lib/ycmd/ycm_extra_conf.py'

模式切换¶

i - 插入模式 v - 选择模式 - 竖排选择模式 esc - 退出插入或者选择模式

移动¶

hjkl 来移动光标 w - 前进1个单词 b - 后退1个单词 % 0 - 一行的初始位置 $ - 一行的最后位置 gg - 跳到第一行 G - 跳到最后一行

操作¶

y 复制 p 粘贴

录制动作¶

q[a-z] - 录制动作并保存到序号a-z中 @[a-z] - 执行序号a-z的动作
