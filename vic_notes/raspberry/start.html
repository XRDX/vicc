Start¶

桌面虽好，使用树莓派，最主要的目的还是为了学习Linux系统的理念，因此不建议使用桌面版本。同时使用桌面版本需要消耗大量的计算，对树莓派来说增加了很多的负担，维持，从一开始，我们就抛弃了树莓派带桌面的版本，转向Lite版本，即没有带桌面的版本。

安装¶

  1. 格式化SD卡
  2. 写入img文件
  3. 在boot中添加一个名为ssh的空文件（为了可以使用ssh来连接）
  4. 用网线连接树莓派到路由器，启动，从路由器中找到树莓派的地址，使用putty通过ssh来访问
  5. 初始用户名pi，密码raspberry

config¶

树莓派自带很多设置功能，我们只要按照树莓派的设置就可以完成很多功能了，而不需要去修改配置文件，大大减少了初始错误。

  1. 添加wifi密码，添加完wifi后，就不需要网线来连接了，通过wifi的ip地址来直接连接。
sudo raspi-config

参考wifi的ip地址：

ifconfig wlan0

install aria2¶

sudo apt-get install -y aria2 ca-certificates

# webui-aria2
wget https://github.com/ziahamza/webui-aria2/archive/master.zip
unzip -qu master.zip
cd webui-aria2-master/
sudo cp docs /var/www/webui-aria2

cd /etc/nginx/sites-avaiable
cat > aria2 <<-EOCFG
server {
        listen 8889 default_server;
        listen [::]:8889 default_server;
        root /var/www/webui-aria2;
        index index.html index.htm index.nginx-debian.html;
        server_name _;
        location / {
                try_files $uri $uri/ =404;
        }
}
EOCFG
cd ../sites-enabled
sudo ln -s ../sites-avaiable/aria2
sudo nginx -t 
sudo nginx -s reload

install docker on raspberry¶

# Install Docker
curl -sSL https://get.docker.com | sh

# Add permission to Pi User to run Docker Commands
sudo usermod -aG docker pi

# Test Docker installation
docker run hello-world

# IMPORTANT! Install proper dependencies
sudo apt-get install -y libffi-dev libssl-dev
sudo apt-get install -y python python-pip

# Install Docker Compose
sudo pip install docker-compose
